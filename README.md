# Esri x NVIDIA Integration Explorer

Interactive web app showcasing all 20 Esri and NVIDIA technology integration points, built with ArcGIS Maps SDK for JavaScript 5.0 and Calcite Design System.

## Live Demo

**[View Live Demo](https://garridolecca.github.io/Nvidia-esri-integrations/)**

## Features

- **Interactive Map** — Color-coded points at relevant locations worldwide (Redlands, Raleigh, Zurich, etc.)
- **Search & Filter** — Filter by status (Production, Emerging, Active, etc.) or search by keyword
- **Detail Panel** — Full use case description, technology tags, and architecture flow diagrams
- **Keyboard Navigation** — Arrow keys to browse, click to select
- **20 Integrations Covered** — From GPU-accelerated spatial analysis to smart city video analytics to climate simulation
